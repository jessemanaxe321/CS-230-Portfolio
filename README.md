# CS-230-Portfolio

This repository is for my CS 230 Operating Platforms coursework at Southern New Hampshire University.

## Course Overview

In this course, I worked with software design concepts related to operating platforms, system architecture, memory management, storage management, distributed systems, networks, and security. The main project focused on The Gaming Room and its game, Draw It or Lose It.

## Project

The main portfolio artifact for this course is a software design document for Draw It or Lose It. This document includes recommendations for expanding the application from an Android-only game into a web-based application that can run across multiple platforms.

## Skills Demonstrated

- Software design documentation
- Operating platform evaluation
- Client-server architecture
- Distributed systems and networking concepts
- Memory and storage management
- Security planning for web-based applications
